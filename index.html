유튜브 썸네일 기획·제작 및 클릭률 개선
2024.07 – 2025.02
브레드이발소의 세계관과 트렌드를 시각적으로 재해석하여 유튜브 클릭률을 높이는 썸네일을 기획·제작
Why
  • 기여도: 썸네일 기획 및 디자인 전 과정 100% 직접 수행
  • 타겟: 1020세대
  • 목적: 시청자의 흥미를 유도하고 CTR을 높이는 썸네일 기획·제작
Problem
  • 기존 썸네일은 반복적인 형식과 제한된 소재로 구성되어 클릭을 유도하는 요소가 부족
  • 콘텐츠 노출은 유지되었지만 CTR, 조회수, 구독자 증가로 이어지지 않는 구조적 한계 존재
Solution
  • 썸네일을 단순 장면 캡처에서 시청자의 호기심을 유도하는 감정·반전 중심 이미지로 전환
  • 시즌 이슈·트렌드 콘텐츠를 반영한 썸네일을 기획하여 시의성과 클릭 유도 요소를 동시에 강화
  • 일부 썸네일에 AI 이미지 생성 방식 도입 — LMArena·Gemini로 이미지 생성 후 Photoshop으로 보정.
    에피소드 캡처만으론 구현할 수 없는 컨셉 실현
Performance
  • 입사 전/후 4개월 비교 기준 CTR 5.7% → 6.0% 상승
  • 예상 수익 $8,500 이상 증가
  • 구독자 증가율 약 12% 향상
Roles & Responsibilities
  • 이벤트·트렌드 이슈 기반 썸네일 기획 — 캐릭터 중심 구조 → 클릭 유도형으로 전환
  • YouTube Studio A/B 테스트를 활용해 썸네일별 CTR 및 시청 시간 성과 비교·검증
  • 콘텐츠 업로드 일정 수립 및 운영 총괄
Lessons Learned
  • 썸네일은 단순한 시각 요소가 아니라 사용자의 클릭을 유도하는 '상황 설계'라는 점을 학습
  • 개인적인 흥미보다 대중이 공감할 수 있는 보편적 상황과 감정을 반영할 때 CTR과 반응이
    더 높아진다는 인사이트 확인

체리툰
체리툰 오리지널 숏폼 콘텐츠 기획·제작
2025.09 – 2026.03
트렌드 기반 숏폼 콘텐츠 전략 전환을 통해 조회수 300만+ 성과를 만든 오리지널 쇼츠 기획·제작
Why
  • 기여도: 기획 및 여러 제작 부서와의 커뮤니케이션 필요 — 70% 기여
  • 타겟: 1020세대
  • 목적: 타겟의 관심을 유도하고 조회수를 확보할 수 있는 숏폼 콘텐츠 기획·제작
Problem
  • 시즌 3 제작 지연으로 신규 콘텐츠 공급이 어려운 상황
    → 기존 재편집 콘텐츠만으로는 사용자 이탈을 막기 어려운 구조
  • 캐릭터 IP의 인지도는 유지되고 있어 지속적인 노출과 팬 유입을 위한 대체 콘텐츠 전략이 필요
Solution
  • 숏폼 중심 콘텐츠 운영 전략으로 전환하고 트렌드·밈·시의성을 반영한 오리지널 콘텐츠 기획
  • 캐릭터 성격 기반 상황극 포맷을 설계하여 반복 제작 가능한 콘텐츠 구조 구축
  • 콘텐츠 소비 → 채널 재방문 → 팬 유입으로 이어지는 팬덤 빌딩 구조를 고려해 설계
Performance
  • 4개월간 조회수 300만+ 숏폼 콘텐츠 6건 제작
  • 시청자 평균 참여율 76.7% → 80.3% 상승
Roles & Responsibilities
  • 트렌드 및 타겟 분석 기반 콘텐츠 기획 방향 수립
  • 조회수·참여율 등 핵심 지표 분석 및 개선 방향 도출
  • 데이터 기반으로 콘텐츠 전략을 반복 개선하며 제작 구조 구축
Lessons Learned
  • 숏폼 콘텐츠는 단순 편집이 아니라 타겟과 상황을 기반으로 설계되어야 한다는 점을 학습
  • 포맷 구조에 따라 조회수와 유입 성과가 달라지며 콘텐츠 기획 단계에서 구조 설계가 핵심이라는 인사이트 도출

Crehoho
감정 기반 체험형 콘텐츠 기획 — How are you Now?
2023.05 – 2023.11
추상적인 감정을 관객이 직접 체험할 수 있는 구조로 설계한 콘텐츠 기획 프로젝트.
콜라주 아트워크 → 가상 전시 시나리오 → 굿즈로 이어지는 경험 확장 구조를 직접 기획·제작
Why
  • 기여도: 전시 주제 선정부터 아트워크 기획·제작까지 전 과정 주도
  • 타겟: 개인의 감정을 공유하고 싶어하는 1020세대
  • 목적: 음악 가사의 감정을 시각화해 개인 경험을 공감 가능한 콘텐츠로 확장
Problem
  • 음악 가사나 개인적 감정은 주관적이라 관객이 즉각적으로 공감하기 어려움
  • 텍스트만으로는 감정의 뉘앙스 전달이 제한되어 단순 정보로 소비될 가능성 존재
Solution
  • 가사 속 상황을 시각적으로 재해석해 누구나 자신의 경험을 대입할 수 있는 이미지로 표현
  • 개인적 감정을 '보이는 장면'으로 전환해 관객이 자신의 경험과 연결할 수 있도록 설계
  • 콘텐츠 → 현장 체험 → 굿즈로 이어지는 경험 확장 구조 기획
Lessons Learned
  • 콘텐츠는 단순히 감각적으로 만드는 것이 아니라, 관객의 경험과 연결될 때 공감을 만든다는 점을 체감
  • 추상적인 감정도 시각화를 통해 보편적인 공감 요소로 확장될 수 있음을 확인

CJ ENM
투니버스 IP 기반 콘텐츠 기획·운영 (썸네일·소셜 콘텐츠)
2021.05 – 2021.11
CJ ENM 투니버스 IP를 활용해 클릭을 만드는 썸네일과 콘텐츠처럼 소비되는 소셜 콘텐츠를 함께 설계.
감정·궁금증 중심 썸네일과 트렌드 기반 캐릭터 콘텐츠를 결합해 유입부터 반응까지 이어지는
콘텐츠 구조를 구축
Why
  • 기여도: 썸네일 및 소셜 콘텐츠 기획·제작 전 과정 직접 수행
  • 타겟: 1020세대
  • 목적: 클릭 → 소비 → 반응으로 이어지는 구조 설계 및 성과 개선
Problem
  • 기존 콘텐츠는 썸네일 클릭 이후 소비와 반응으로 이어지지 않는 구조적 한계 존재
  • 홍보성 콘텐츠는 광고 느낌이 강해 자연스럽게 소비되지 못하고 이탈로 이어지는 문제 발생
Solution
  • 썸네일은 감정·궁금증 중심으로 클릭 유도, 소셜 콘텐츠는 트렌드 기반 상황극으로 자연스러운 소비 유도
  • 유입과 소비 각 단계에서 사용자 반응을 유도하는 구조로 개선
Roles & Responsibilities
  • 감정·궁금증 중심 썸네일 기획으로 클릭 유도 개선
  • 캐릭터 상황극 + 트렌드 반영 소셜 콘텐츠 제작으로 자연스러운 소비 유도
  • 역할별 전략 분리를 통해 클릭과 반응을 각각 최적화
Lessons Learned
  • 개인 취향보다 대중이 공감할 수 있는 상황·감정을 시각화하는 것이 중요함을 체감
  • 콘텐츠는 적절한 타이밍과 포인트 설계, 반복 노출을 통해 성과로 이어진다는 인사이트 도출
